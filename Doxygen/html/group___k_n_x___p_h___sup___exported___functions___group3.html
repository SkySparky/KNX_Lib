uint8_t KNX_Ph_Reset_request ( void  )

Request to reset the KNX Physical Layer module and waiting for the Reset_indication.

Return values
Errorcode, See Physical Layer Error Code.

Send Ph_Reset request.

If encounter a problem, return PH_ERROR_REQUEST

Waiting for the Reset_indication.

If receive the response, set state to PH_RESET.

Return PH_ERROR_NONE.

Else return PH_ERROR_RESPONSE.

uint8_t KNX_Ph_State_request ( void  )

Requests the internal communication state from the TP-UART-IC and waiting for the State_indication.

Return values
Errorcode, See Physical Layer Error Code.

Send Ph_Reset request.

If encounter a problem, return PH_ERROR_REQUEST

Waiting for the State_indication.

If receive the response, set state to PH_NORMAL.

Return PH_ERROR_NONE.

Else return PH_ERROR_STATE.

Else return PH_ERROR_RESPONSE.
