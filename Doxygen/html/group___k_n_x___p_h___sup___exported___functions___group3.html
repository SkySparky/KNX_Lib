Request to reset the KNX Physical Layer module and waiting for the Reset_indication.

Return values
Errorcode, See Physical Layer Error Code.

Send Ph_Reset request.

If encounter a problem, return PH_ERROR_REQUEST

Waiting for the Reset_indication.

Else return PH_ERROR_RESPONSE.
